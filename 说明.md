# Hugo 项目目录结构

- **archetypes/**: 包含新建内容时的模板文件。
- **assets/**: 存放需要被 Hugo Pipes 处理的资源（如 SCSS, JS）。
- **content/**: 存放网站的 Markdown 内容文件。
- **data/**: 存放 Hugo 在生成网站时可以使用的配置文件（JSON, YAML, TOML）。
- **layouts/**: 存放用于渲染内容的 HTML 模板。
- **static/**: 存放静态资源（图片、CSS、JS），这些文件会原封不动地复制到发布目录。
- **themes/**: 存放下载或自定义的主题。
- **config.toml / hugo.toml**: 项目的核心配置文件。
- **public/**: 执行 `hugo` 命令后生成的静态网站文件（默认不包含在源码库中）。
