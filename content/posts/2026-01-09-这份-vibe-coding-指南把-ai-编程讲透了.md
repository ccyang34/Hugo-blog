## 描述

最近在 GitHub 上翻到这个 Vibe Coding 指南，6.2k stars，强烈推荐你也看看。

它把 AI 编程这件事讲得特别系统。从最基础的网络环境配置、开发环境搭建，到进阶的胶水编程、白板驱动开发、AI蜂群协作，每一步都有详细文档。

它的道·法·术方法论也很有意思，“道”讲思维原则，“法”讲拆解策略，“术”讲具体操作。三层递进，把AI编程从碰运气变成有章法。

最值得收藏的是它的提示词库，分成元提示词、系统提示词、编程提示词三类，直接复制就能用。还有一个在线表格收录了几百个精选提示词，省得自己摸索了。

这个教程适合已经用 ChatGPT 或 Claude 写过代码、但效率总上不去的朋友。零基础也能跟着入门教程一步步来。
