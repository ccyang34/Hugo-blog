**作者**: AI范儿

★ 设为星标 | 只讲人话，带你玩转AIGC。这几天 Clawdbot 太火了，但我看了很多教程都是对接 Telegram、WhatsApp 之类的。

有点恼火，我又不用这些玩意，难道就不能直接对接飞书吗？

研究了半天，终于搞定了，下面这个保姆教程，一步步跟着做，你应该也可以搞定。

开始之前先给大家捋一下需要准备什么：

一台电脑就可以了，不用买什么 Mac Mini、也不用买 VPS 啥的。

我在自己的 MacBook 上面安装，这很不安全，所以建议你不要用日常工作Mac， 也不要用常用的Apple id，使用前备份下数据。

流程包括几步：

1）安装 Clawdbot

2）配置飞书（这步最麻烦）

3）安装飞书插件

话不多说，我们直接开始。

**01｜安装 Clawdbot**

这一步其实非常简单，先从一条命令开始，在 Mac 上打开你的终端（Teminal 或者 iTerm 之类的），然后黏贴下面的命令：

* 


```
curl -fsSL https://clawd.bot/install.sh | bash
```

**02 配置飞书**

配置飞书大概要分三个方面步骤：

1）添加一个机器人应用

2）给这个机器人应用添加功能：比如可以接收信息

3）给这个机器人应用添加权限：比如发送接收消息什么的

先第一步：进入飞书应用中心：

https://open.feishu.cn/app?lang=zh-CN

![Image](https://i0.wp.com/mmbiz.qpic.cn/mmbiz_png/OdmYSS49h7HCLQFPLhpdcE8qUWwdxBsMFBuVcibnzzWDJ5JH8KYwqITWPVHgfKE0bgRuNf1qeRo2q3AaibLCwmsQ/640?wx_fmt=png&from=appmsg)图：传创建企业自建应用创建一个企业自建应用

![Image](https://i0.wp.com/mmbiz.qpic.cn/mmbiz_png/OdmYSS49h7HCLQFPLhpdcE8qUWwdxBsMv6I4nucWzsTu2KGWnhQrdXI432Wwh6jrEiciaN5OmJDx1DbYzCic1XRnA/640?wx_fmt=png&from=appmsg)图：创建企业自建应用应用名和描述随便写就好了。

因为我们希望在飞书里面添加一个可以对话的机器人，只不过这个机器人会把消息发给 Clawdbot，所以先得加机器人：

添加机器人：在左侧的“添加应用能力”中添加“机器人”

![Image](https://i0.wp.com/mmbiz.qpic.cn/mmbiz_png/OdmYSS49h7HCLQFPLhpdcE8qUWwdxBsMBYEuF9qBiabrkyjK4KAibK0hyicBSpWRy9ocl3SyQLr7UOfnPN5c9cE3A/640?wx_fmt=png&from=appmsg)图：添加机器人注意，创建完之后点顶部的“创建版本”先发布一下，版本可以发布多次。接下来，为机器人添加“接收消息”的能力。

在“事件与回调”里面“添加事件”，然后选择“消息与群组”里面的“接收消息”（缺了这一步，你给机器人发消息会没有回复）

![Image](https://i0.wp.com/mmbiz.qpic.cn/mmbiz_png/OdmYSS49h7HCLQFPLhpdcE8qUWwdxBsM0Sl3GGxtKoBVIhF5icF4YicOheBeo6c8pdrpK5hGvNbtmRf0BPtxQqYw/640?wx_fmt=png&from=appmsg)图：添加事件设置“回调配置”，订阅方式选“使用长连接”（如果前面没有发布机器人版本，这里会报错）

![Image](https://i0.wp.com/mmbiz.qpic.cn/mmbiz_png/OdmYSS49h7HCLQFPLhpdcE8qUWwdxBsMRo6ZQ4ONib23etT2PKpG7GEwrMDcyWd6jahqq3L5X3DKQmFG0k30ApQ/640?wx_fmt=png&from=appmsg)图：使用长链接能力添加好了，接下来还要为它开通权限：

![Image](https://i0.wp.com/mmbiz.qpic.cn/mmbiz_png/OdmYSS49h7HCLQFPLhpdcE8qUWwdxBsM5WdCAyjz4j1hknm6sx2iaqNMTdYdjYcyiac05BWNVWtGVs2LF1NtFcrw/640?wx_fmt=png&from=appmsg)图：为机器人开通权限记得要开通下面这些权限：

![Image](https://i0.wp.com/mmbiz.qpic.cn/mmbiz_png/OdmYSS49h7HCLQFPLhpdcE8qUWwdxBsM3veFyNshg6N5SttdDoO1lErvOGVTUHXXyZRicDzO2sGBzkGOjmeRmiag/640?wx_fmt=png&from=appmsg)图：需要开通的权限可以在开通权限的弹出框里面直接搜索。

为这个机器人做了这么多配置，最终需要发布所有这些修改才行，最后继续选择最顶部的“创建版本”

![](https://i0.wp.com/mmbiz.qpic.cn/mmbiz_png/OdmYSS49h7HCLQFPLhpdcE8qUWwdxBsMDqcqaGdPPO9pUX7lIa43JJibEibjJ9oZuI7N4WlyJcic0RAUL34phO8RQ/640?wx_fmt=png&from=appmsg)然后输入一个版本号和更新说明，之后点“保存”就好了。

**03 配置飞书**

现在 Clawdbot 和飞书都单独设置好了，他们最终还是需要连接起来才行。

我们继续打开一个终端（terminal），然后安装飞书插件：

* 


```
clawdbot plugins install @m1heng-clawd/feishu
```
之后就是配置飞书的参数：

* 
* 
* 


```
clawdbot config set channels.feishu.appId "cli_xxxxx"clawdbot config set channels.feishu.appSecret "your_app_secret"clawdbot config set channels.feishu.enabled true
```
注意这里的 appID 和 appSecret 要在刚才的飞书后台获取：

![Image](https://i0.wp.com/mmbiz.qpic.cn/mmbiz_png/OdmYSS49h7HCLQFPLhpdcE8qUWwdxBsMiblBAB5B4hrLYWZCb1NCd95a4Ze2R12K4ZCB7tVWxwoCrrekKTpVD8Q/640?wx_fmt=png&from=appmsg)![](https://i0.wp.com/mmbiz.qpic.cn/mmbiz_png/OdmYSS49h7HCLQFPLhpdcE8qUWwdxBsMnbRbsETs2fXEqSoO40nVTcc5w4H3Xfk0Q4NWsL3AiaS1wm06BW7lA5Q/640?wx_fmt=png&from=appmsg)**04 重启**

所有的都搞定了，现在记得重启一下 clawdbot：

* 


```
clawdbot gateway restart
```
**05 使用**

不管是手机还是电脑，直接在飞书里搜你刚才起的机器人名字。

给它发个“你好”，如果它回你了，恭喜你，你的专属 AI 助理正式上岗了！

你甚至可以把它拉进群里，直接 @ 它帮你写文档、出方案。
